This may be needed if you get an error using the GMaps API key without my debug certificate

- Place the "debug.keystore" in your "%USERPROFILE%\.android" folder.

If you care to view the information in it, I made a bat file to go along with it.
	- Place the bat file anywhere, and run it.
	- It will prompt for password, the default is "android"
	- It will display a few certificates, the one I use is the "SHA1" cert.
